# NexusBot — Rule-Based AI Chatbot

A rule-based AI chatbot built in Python as part of the **DecodeLabs AI Internship — Project 1 **.

## What it does
- Responds to 30+ predefined intents using dictionary-based O(1) lookup
- Covers AI, ML, Python, Data Structures, career advice, and more
- Sanitizes input (case & whitespace handling)
- Greets user by name with time-aware messages
- Runs in a continuous loop with a clean exit command

## Tech Stack
- Python 3.x
- Zero external libraries

## How to Run
```bash
python chatbot.py
```

## Concepts Demonstrated
- Control flow & decision-making logic
- Input sanitization
- Dictionary (Hash Map) over if-elif ladder — O(1) vs O(n)
- IPO Model (Input → Process → Output)
- Infinite loop with kill command
